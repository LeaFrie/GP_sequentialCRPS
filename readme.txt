This is the code associated with the manuscript 
'CRPS-Based Targeted Sequential Design with Application in Chemical Space' 
submitted to the Journal of Computational and Graphical Statistics.

- Data: original Photoswitch dataset, and a synthetic version thereof

Thawani, A. R., Griffiths, R.-R., Jamasb, A., Bourached, A., Jones, P., McCorkindale,
W., Aldrick, A. A., and Lee, A. A. (2020). The Photoswitch Dataset: A molecular
machine learning benchmark for the advancement of synthetic chemistry. ChemRxiv,
doi:10.26434/chemrxiv.12609899.v1. This content is a preprint and has not been peer-reviewed.

- Sequential: code for GP modelling and sequential design
- Plots: for results and illustrations
